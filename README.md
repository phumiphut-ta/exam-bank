# exam-bank (ระบบคลังข้อสอบและพิมพ์ข้อสอบ A4 แบบออฟไลน์)

ระบบบริหารจัดการคลังข้อสอบออฟไลน์ 100% บันทึกข้อมูลปลอดภัยไว้ภายในเครื่องเบราว์เซอร์ของผู้ใช้ด้วย IndexedDB พร้อมฟังก์ชันการจัดพิมพ์กระดาษคำถามและกระดาษเฉลยจัดหน้าแยกกันแบบ A4 อย่างสวยงามระดับพรีเมียม

---

## ✨ คุณสมบัติหลัก (Key Features)

* **📦 คลังข้อสอบออฟไลน์ (Offline Question Bank):** เพิ่ม แก้ไข ลบ ข้อสอบและหมวดหมู่ได้ไม่จำกัด บันทึกข้อมูลและรูปภาพในเครื่องของผู้ใช้ด้วย IndexedDB ไม่ผ่าน Server
* **🖼️ แนบรูปภาพประกอบโจทย์:** สามารถอัปโหลดรูปภาพประกอบโจทย์ข้อสอบได้ โดยระบบจะทำการบีบอัดและแปลงรูปภาพเป็น Base64 โดยอัตโนมัติ
* **🔍 ตรวจสอบคำถามซ้ำ (Duplicate Checking):** แจ้งเตือนเมื่อโจทย์ซ้ำทันทีขณะพิมพ์ พร้อมกล่องแจ้งยืนยันก่อนกดบันทึก
* **🎲 จัดชุดข้อสอบสุ่มอัจฉริยะ (Exam Generator):** 
  * สุ่มเลือกข้อสอบตามสัดส่วนจำนวนข้อที่กำหนดในแต่ละหมวดหมู่
  * **สุ่มสลับตำแหน่งข้อสอบ (Shuffle Questions)**
  * **สุ่มสลับตำแหน่งตัวเลือก ก ข ค ง (Shuffle Options)** โดยระบบจะทำการปรับเปลี่ยนคำตอบของเฉลยให้ตรงตามตำแหน่งที่สลับล่าสุดให้อย่างแม่นยำอัตโนมัติ
* **📄 จัดพิมพ์ข้อสอบและเฉลยแยกหน้า A4:**
  * **กระดาษคำถาม (Question Sheet):** มีการจัดหน้า A4 สวยงาม แสดงหัวข้อสอบ คำชี้แจง และช่องกรอกข้อมูลผู้สอบ (ชื่อ-นามสกุล, ชั้น, เลขที่, รหัส) รวมอยู่ในบรรทัดเดียว พร้อมรันเลขหน้า `หน้า X จาก Y` ทุกหน้า
  * **กระดาษเฉลย (Answer Key):** พิมพ์เฉลยแยกจากข้อสอบโดยสิ้นเชิง โดยจัดหน้าแบบคอลัมน์ย่อกระดาษหลายแถวเพื่อความหนาแน่นและประหยัดกระดาษพิมพ์เฉลย
  * **Interactive Preview:** ผู้ใช้สามารถคลิกแก้ไขข้อความหัวข้อสอบ คำชี้แจง และช่องผู้สอบได้สดๆ บนตัวอย่างหน้ากระดาษ A4 ก่อนพิมพ์จริง
* **💾 สำรองและกู้คืนข้อมูล (Backup & Restore):** ส่งออกข้อมูลทั้งหมดเป็นไฟล์ `.json` เพื่อสำรองเก็บไว้หรือย้ายไปทำงานคอมพิวเตอร์เครื่องอื่น

---

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

* **Frontend:** HTML5, CSS3, JavaScript (ES6+ Vanilla)
* **Fonts:** Outfit (UI), Sarabun (Exam/Thai font)
* **Database:** IndexedDB (ระบบฐานข้อมูลออฟไลน์ระดับเว็บบราวเซอร์)
* **Icons:** SVG Inline (Feather/Lucide Icons style)

## 🚀 วิธีเปิดใช้งานภายในเครื่อง (How to Run)

เปิดใช้งานได้ง่ายๆ 4 ทางเลือก (อ่านแบบละเอียดได้ที่ [วิธีเปิดใช้งาน.md](./วิธีเปิดใช้งาน.md)):
1. **ดับเบิ้ลคลิกไฟล์:** ดับเบิ้ลคลิกไฟล์ `index.html` เพื่อรันผ่านเบราว์เซอร์ทันที
2. **คำสั่ง Node.js:** รันคำสั่ง `npm run dev` (จำลองพอร์ต `http://localhost:8080`)
3. **คำสั่ง Python:** รันคำสั่ง `python -m http.server 8080`
4. **ติดตั้งเป็น Desktop App:** ติดตั้งผ่านบราวเซอร์ Chrome หรือ Edge (สัญลักษณ์ Install App 📥 ตรงช่อง URL)

## 📤 วิธีอัปโหลดโครงการขึ้น GitHub (How to Upload to GitHub)

หากคุณติดตั้งเครื่องมือ Git ไว้บนคอมพิวเตอร์ของคุณแล้ว สามารถอัปโหลดโครงการนี้ขึ้น GitHub ได้ตามขั้นตอนต่อไปนี้:

1. **สร้าง Repository บน GitHub:**
   - เข้าเว็บ [GitHub](https://github.com/) และล็อกอินเข้าสู่ระบบ
   - คลิกปุ่ม **"New"** (หรือ "Create Repository") เพื่อสร้างคลังเก็บโค้ดใหม่
   - ตั้งชื่อคลังเก็บโค้ด เช่น `exam-bank`
   - เลือกเป็น **Public** หรือ **Private** ตามต้องการ
   - **ไม่ต้องติ๊ก** เลือก Add a README, Add .gitignore หรือ Choose a license (เนื่องจากโครงการนี้มีอยู่แล้ว)
   - คลิก **"Create repository"**

2. **เชื่อมต่อและพุชโค้ดขึ้น GitHub (พิมพ์คำสั่งใน Terminal/cmd ในโฟลเดอร์โครงการ):**
   *(ระบบได้ทำการ `git init` และ Commit แรกให้เบื้องต้นเรียบร้อยแล้วในเครื่องของคุณ)*
   
   เปิด Command Prompt ในโฟลเดอร์โครงการนี้ แล้วพิมพ์คำสั่ง:
   ```bash
   # 1. เชื่อมต่อ Git ท้องถิ่นเข้ากับ GitHub Repository ของคุณ (เปลี่ยน URL ให้เป็นของคุณเอง)
   git remote add origin https://github.com/ชื่อผู้ใช้ของคุณ/exam-bank.git
   
   # 2. ตั้งชื่อกิ่งหลักเป็น main
   git branch -M main
   
   # 3. ส่งข้อมูลทั้งหมดขึ้น GitHub
   git push -u origin main
   ```
