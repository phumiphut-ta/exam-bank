# คู่มือการเปิดใช้งานระบบคลังข้อสอบ (exam-bank) แบบทั่วไป
*(ในกรณีที่ต้องการรันด้วยตัวเองภายนอก Antigravity หรือแชร์โฟลเดอร์ให้ผู้อื่นใช้งาน)*

เนื่องจากระบบคลังข้อสอบนี้ได้รับการพัฒนาในรูปแบบ **Single Page Application (HTML/CSS/JS)** ร่วมกับฐานข้อมูลฝั่งไคลเอนต์ **IndexedDB** ทำให้คุณสามารถรันใช้งานได้ทันทีบนคอมพิวเตอร์ทั่วไปโดยไม่ต้องทำการตั้งค่าเซิร์ฟเวอร์หรือฐานข้อมูลภายนอกให้ยุ่งยาก 

โดยมีวิธีเปิดใช้งานให้เลือกตามความสะดวก 4 ช่องทางดังนี้ครับ:

---

## ⚡ วิธีที่ 1: ดับเบิ้ลคลิกเปิดใช้งานโดยตรง (ง่ายที่สุด - ไม่ต้องลงโปรแกรมเพิ่ม)
วิธีนี้สะดวกที่สุด สามารถนำโฟลเดอร์ `exam-bank` ไปคัดลอกลงแฟลชไดรฟ์หรือส่งต่อให้เพื่อนร่วมงานเปิดใช้งานบนเครื่องใดก็ได้ทันที
1. เปิดโฟลเดอร์โครงการ `exam-bank`
2. ดับเบิ้ลคลิกที่ไฟล์ **`index.html`**
3. ระบบจะเปิดขึ้นบนเว็บเบราว์เซอร์หลักในเครื่องของคุณ (แนะนำ Google Chrome, Microsoft Edge หรือ Safari) และพร้อมใช้งานทันที
   * *หมายเหตุ:* ข้อมูลข้อสอบและรูปภาพทั้งหมดจะบันทึกอยู่ในฐานข้อมูลในบราวเซอร์ของคอมพิวเตอร์เครื่องนั้นๆ แบบออฟไลน์ 100% ปลอดภัยและข้อมูลไม่สูญหายแม้จะปิดบราวเซอร์ไป

---

## 💻 วิธีที่ 2: รันผ่านคำสั่ง Node.js / npm (มาตรฐานนักพัฒนา)
เหมาะสำหรับเครื่องที่มีการติดตั้ง **Node.js** อยู่แล้ว และต้องการจำลองเซิร์ฟเวอร์ภายในเครื่อง
1. เปิดโปรแกรม **Terminal** หรือ **Command Prompt (cmd)**
2. ใช้คำสั่งย้ายเข้าไปยังโฟลเดอร์ `exam-bank` (เช่น `cd path/to/exam-bank`)
3. พิมพ์คำสั่งด้านล่างนี้เพื่อรันระบบ:
   ```bash
   npm run dev
   ```
4. ระบบจะจำลองเซิร์ฟเวอร์ขึ้นที่พอร์ต 8080 และเปิดเบราว์เซอร์ไปที่ลิงก์ **`http://localhost:8080`** ให้โดยอัตโนมัติ

---

## 🐍 วิธีที่ 3: รันผ่านคำสั่ง Python (ทางเลือกเสริม)
เหมาะสำหรับคอมพิวเตอร์ที่มี **Python** ติดตั้งอยู่แล้วในระบบ
1. เปิด **Command Prompt (cmd)** หรือ **Terminal** 
2. ย้ายเข้าไปในโฟลเดอร์ `exam-bank`
3. พิมพ์คำสั่งรันเซิร์ฟเวอร์จิ๋วของ Python:
   ```bash
   python -m http.server 8080
   ```
4. เปิดเว็บเบราว์เซอร์แล้วเข้าไปที่ลิงก์ **`http://localhost:8080`**

---

## 📱 วิธีที่ 4: การติดตั้งเป็นแอปพลิเคชันลงบนเครื่องคอมพิวเตอร์ (PWA)
คุณสามารถติดตั้งแอปนี้ลงบน Desktop หน้าจอคอมพิวเตอร์ของคุณเพื่อเปิดใช้งานเหมือนโปรแกรมสำเร็จรูปทั่วไปได้ง่ายๆ:
1. เปิดระบบด้วย **Google Chrome** หรือ **Microsoft Edge** (ผ่านวิธีที่ 2 หรือ 3)
2. สังเกตที่ช่องกรอก URL ด้านบนขวา จะมีสัญลักษณ์ **รูปจอคอมพิวเตอร์พร้อมเครื่องหมายบวก 📥 (Install App)** ปรากฏขึ้นมา
3. คลิกที่สัญลักษณ์นั้นแล้วกด **"ติดตั้ง" (Install)**
4. ระบบจะทำการสร้างไอคอนแอปพลิเคชันคลังข้อสอบไว้บนหน้าจอเดสก์ท็อปของคุณ สามารถดับเบิ้ลคลิกเปิดใช้งานแบบแยกหน้าต่างเดี่ยวไม่มีแถบบราวเซอร์กวนใจได้ทันที!

---

### 💾 คำแนะนำสำหรับการสำรองข้อมูล (Backup)
เนื่องจากระบบบันทึกข้อมูลไว้ในเบราว์เซอร์ของเครื่องนั้นๆ หากคุณต้องการย้ายคอมพิวเตอร์ในการทำงาน หรือต้องการล้างประวัติบราวเซอร์:
1. ให้ไปที่แถบเมนู **"สำรอง/นำเข้าข้อมูล"** ในระบบคลังข้อสอบ
2. กดปุ่ม **"ส่งออกข้อมูลสำรอง (.json)"** เพื่อเซฟไฟล์สำรองข้อสอบทั้งหมดเก็บไว้
3. เมื่อไปเปิดใช้งานในคอมพิวเตอร์เครื่องใหม่ หรือบราวเซอร์อื่น ให้เข้าไปที่เมนูเดียวกัน และเลือกไฟล์ JSON นั้นเพื่อกด **"นำเข้าข้อมูลคลังข้อสอบ"** ข้อสอบและภาพทั้งหมดจะกลับมาเหมือนเดิมทันที
